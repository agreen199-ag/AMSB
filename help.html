External CRM Tracker

This report contains a list of external relationships.

At the top of the report are filters for searching the list, including Military Branch, Key POC, City, Organization Type, Focus area, Operation Type, and Relationship Owner.

The list of data is in the center of the report.

At the bottom of the report is a map showing the location of the Branch.

Daily Tracker (Onboarding)

This report contains information on candidates sorted by their cohort number.

The cohort number can be selected with the dropdown in the top left of the report. Next to this are several other filters including Last Name, First Name, Key POC, Site, Pillar, and Role

The data for each cohort member can be seen at the bottom of the report.

Section 1 - Overview

This report gives a general overview of AMSB candidates.

The statistics for percent applied, enrolled, completed, job interviews, and job offers can be found at the top.

The pie charts give a visual representation of this data. Percent applied, enrolled, and completed are in the left "Application Status" chart, while percent interviewed, job offered, and onboarded are in the right "Onboarding Status" chart.

You can filter the percentages and pie charts by site by using the check boxes in the top left.


Section 2 - Candidate Table

This report contains a detailed list of AMSB candidates.

The table itself is on the right. Each row represents a candidate, and each column holds various data for that candidate. To see all of the data columns, use the horizontal scroll bar at the bottom of the table.

To filter the table by site, role, application status, or onboarding status, use the check boxes on the left.

To filter the table by last or first name, type the name to search for in the text boxes on the left and press enter.

To filter the table by application date, days until separation date, or days until start date, use the sliders on the left. You can either change the lower and upper bounds by typing them in the number box and pressing enter, or by dragging the circles on the slider.

The map at the bottom displays candidate locations. Clicking on a candidate in the table will show their location in the map, and clicking on a dot in the map will show all candidates in that location.
